# demo-topics
A demo created with React.  
A word cloud that displays the topics in the topics.json
  
## INSTALLATION

###GLOBAL GULP
```javascript
npm install -g gulp
```
### DEPENDENCES
```javascript
npm install
```
### RUN DEV
```javascript
gulp
```
### RUN TESTS
```javascript
npm test
```
### RELEASE
```javascript
gulp release
```
After running this task in the folder "dist" you will have your production source
